⚠️ Restaurar Dados Padrão

Restaura cada seção para os dados originais do sistema. Use apenas se quiser desfazer todas as edições de uma seção específica. Esta ação não pode ser desfeita.

☁️ Integração com Cloudinary

Configure o Cloudinary para que as imagens da Galeria sejam hospedadas na nuvem e fiquem visíveis em todos os dispositivos. O plano gratuito suporta até 25 GB — mais do que suficiente.

📋 Como configurar:
1. Acesse cloudinary.com → crie uma conta gratuita
2. No Dashboard, copie o Cloud name
3. Vá em Settings → Upload → Add upload preset → defina como Unsigned → salve e copie o nome do preset
🚀 Publicar no GitHub Pages

Como o site é estático, outros usuários não veem os dados salvos aqui. Clique em Exportar data.js para gerar o arquivo com todo o conteúdo atual (avisos, regras, descarte, configurações) como padrão do site. Depois substitua js/data.js no projeto e faça git push — todos passarão a ver o mesmo conteúdo.

⚠️ Imagens da galeria enviadas por upload (base64) são grandes demais para o arquivo. Coloque os arquivos de imagem na pasta img/ do projeto e use o campo "Caminho da imagem" no admin da galeria para referenciá-las pelo nome (ex: img/piscina.jpg).
