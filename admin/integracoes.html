☁️ Cloudinary — Hospedagem de Imagens

Armazena as imagens da Galeria e o plano de fundo na nuvem, tornando-os visíveis em todos os dispositivos. O plano gratuito suporta até 25 GB.

📋 Como configurar:
1. Acesse cloudinary.com e crie uma conta gratuita
2. No Dashboard copie o Cloud name
3. Vá em Settings → Upload → Add upload preset → defina como Unsigned → salve e copie o nome do preset
🚀 Publicar no GitHub Pages

Como o site é estático, outros usuários não veem os dados salvos aqui. Clique em Exportar data.js para baixar o arquivo com todo o conteúdo atual como padrão do site. Depois substitua js/data.js no projeto e faça git push.

⚠️ Imagens da galeria enviadas em base64 são muito grandes para o arquivo. Use o Cloudinary (acima) ou adicione as imagens na pasta img/ e referencie pelo caminho (ex: img/piscina.jpg).
